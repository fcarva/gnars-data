# Governance Model

## Purpose

This repository should make governance legible across every Gnars era:

- Snapshot V1
- Legacy Ethereum governance
- Base / current `gnars.com` governance

## Proposal taxonomy

Use the following tags consistently in `data/proposals.json` and future proposal templates:

- `governance`
- `operations`
- `analytics`
- `data-infrastructure`
- `docs`
- `product`
- `community`
- `public-goods`
- `culture`
- `marketing`
- `treasury`

## Governance questions this repo should answer

- What kinds of proposals pass?
- How much has the DAO allocated by category?
- Which members are consistently active in governance?
- What work was promised versus what shipped?
- How has the DAO evolved from legacy to current infrastructure?
